# Pattern Generator And Number Analyzer

A beginner-friendly Python project that can generate star patterns and analyze numbers in a range.

---

# Features

## Pattern Generator

Generate different types of patterns:

1. Right Triangle Pattern
2. Reverse Triangle Pattern
3. Pyramid Pattern

## Number Analyzer

Analyze numbers in a given range:

- Check Even and Odd numbers
- Calculate:
  - Sum of Odd Numbers
  - Sum of Even Numbers
  - Sum of All Numbers
