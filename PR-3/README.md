## 📌 Overview

The **Student Data Organizer** is an interactive, menu-driven Python console application designed to manage a collection of student records. It leverages advanced Python 3.10+ features like the `match-case` statement and thoroughly demonstrates the practical application of core collection data types: **Lists, Tuples, Sets, and Dictionaries**.

This project is designed to:
- Strengthen understanding of mutability vs. immutability.
- Practice CRUD (Create, Read, Update, Delete) operations using Dictionaries.
- Implement robust infinite `while` loops and `match-case` menu routing.
- Apply intermediate concepts like string formatting (f-strings) and type casting.

---

## 🎯 Problem Statement

> **Objective:** Create a Python program that manages a collection of student records while applying intermediate-level data structure concepts.

The program must present a continuous menu allowing the user to add, display, update, and delete students. It must use specific data types for specific purposes, ensuring data integrity.
